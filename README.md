# Chest-CT-XAI
Ct Scan of lung images.

Dataset used: https://www.kaggle.com/datasets/c395fb339f210700ba392d81bf200f766418238c2734e5237b5dd0b6fc724fcb

Since the dataset was huge we used small subset of it.

Reference of research papers: https://pmc.ncbi.nlm.nih.gov/articles/PMC11382209/pdf/main.pdf
                              https://www.sciencedirect.com/science/article/pii/S0010482522003791?ref=pdf_download&fr=RR-2&rr=9e2623031b7358b8

Images reference:
acam,azrise,alime,aocul are the images which had annotations by radiologists and then the respective XAI methods have been applied to them.

while lime.png,rise.png,gradcamm.png are XAI methoda applied to random images from the dataset.
